# SSR
SSR DEMO 저장소

## 설치 및 수행
```sh
# server 디렉터리로 이동
cd server 

# server 관련 패키지 설치
npm i -D

# server 실행
npm start
## listening on 8080

## 브라우저에 localhost:8080 접속
```
